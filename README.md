# Lounge Café Website

A modern, responsive café website built using HTML, CSS, and JavaScript.

## Features

- Responsive design
- Hero landing section
- About section
- Interactive menu
- Image gallery with lightbox
- Customer testimonials slider
- Newsletter subscription form
- Mobile navigation
- Smooth scrolling

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Design

## Project Structure

```
css/
images/
js/
index.html
```

## Getting Started

1. Clone the repository.
2. Open `index.html` in your browser.
